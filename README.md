# StockFlow — Sistem Manajemen Gudang

StockFlow adalah aplikasi manajemen gudang berbasis web yang dirancang untuk memudahkan pengelolaan stok, produk, pembelian, dan supplier dalam satu platform terpusat.

> Proyek portfolio yang mendemonstrasikan REST API production-style dengan dashboard React modern.

---

## Teknologi yang Digunakan

**Backend**
- Node.js & Express.js — REST API server
- JWT Authentication — proteksi route berbasis token
- Arsitektur berbasis Middleware — autentikasi, validasi, dan error handling

**Frontend**
- React + Vite — SPA berbasis komponen
- React Router — navigasi antar halaman
- Axios — komunikasi dengan API
- CSS per komponen — styling terpisah tiap halaman

---

## Fitur

- Manajemen Produk — tambah, edit, dan kategorikan produk
- Kontrol Stok — pemantauan stok real-time dengan notifikasi stok minimum
- Mutasi Stok — pencatatan setiap stok masuk dan keluar
- Purchase Order — kelola pemesanan dari supplier
- Kategori & Supplier — organisasi produk dan data vendor
- Manajemen Pengguna — akses berbasis role (Admin)
- Dashboard — nilai inventaris, peringatan stok rendah, dan grafik stok
- Export Laporan — ekspor data dashboard

---

## Halaman

| Halaman | Deskripsi |
|---|---|
| Dashboard | Ringkasan statistik inventaris, grafik stok, dan mutasi terbaru |
| Products | Kelola daftar dan detail produk |
| Stock | Pantau dan perbarui stok saat ini |
| Purchase Orders | Buat dan lacak purchase order |
| Categories | Kelola kategori produk |
| Suppliers | Kelola data supplier |
| Users | Manajemen pengguna oleh Admin |
| Login | Halaman autentikasi |
